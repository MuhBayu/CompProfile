TUJUAN : 
Untuk mempromosikan dan memasarkan sebuah jasa untuk pembuatan software dalam Bisnis IT dan menyediakan infrastruktur dalam bidang IT.

LATAR BELAKANG :
Karena dengan menggunakan situs website sangat mudah dijangkau dan lebih menarik untuk dilihat siapa saja, maka dibuatkanlah sebuah web dinamis yang mencakup pemasaran di bidang IT Business.

FITUR-FITURNYA :
Terdapat menu navigasi diantaranya : 'Home', 'Proyek' dan 'Kontak'
	- HOME adalah tampilan awal beserta ringkasan tentang apa saja yang ada di dalam perusahaan ini serta visi/misi perusahaan.
	- PROYEK adalah tampilan yang berisi proyek-proyek yang sudah dibuat dan dikerjakan oleh perusahaan ini.
	- KONTAK adalah tampilan yang menampilkan kontak perusahaan atau bisa juga mengirim pesan atau pertanyaan melalui form yang tersedia.
Terdapat ADMIN Panel dengan menu diantaranya :
	- DASHBOARD untuk melihat data/statistik jumlah data.
	- JABATAN untuk melihat, menambahkan, mengubah dan menghapus data jabatan.
 	- DIVISI untuk melihat, menambahkan, mengubah dan menghapus data divisi.
	- PROYEK untuk melihat, menambahkan, mengubah dan menghapus data proyek.
	- KARYAWAN untuk melihat, menambahkan, mengubah dan menghapus data karyawan.
	- USERS untuk melihat, menambahkan, mengubah dan menghapus data pengguna.
	- PROFIL PERUSAHAAN untuk mengubah data profil perusahaan.
Terdapat USER Panel dengan menu diantaranya :
	- DASHBOARD untuk melihat data profil pribadi.
	- PROYEK TERLIBAT untuk melihat data proyek yang terlibat dengan pengguna.
 	- PENGATURAN untuk mengubah data informasi pengguna.
